# <img src="https://raw.githubusercontent.com/multiverseweb/Eclipse/main/src/icons/eclipse.png" height="30px"> Eclipse

A beautiful, customizable night mode extension for your browser. Eclipse doesn't just invert colors—it injects a highly optimized dark theme with native support for multiple palettes, custom CSS, and seamless integration with complex web apps like Google Sheets.

---

## Features

- **Multiple Built-In Themes**: Choose from Default Dark, OLED Black, GitHub Dark, Tokyo Night, and Sepia.
- **Custom CSS Engine**: Write your own CSS rules and apply them in real-time to any website.
- **Custom Color Picker**: Tweak background, text, link, and accent colors manually via the extension's sleek neomorphic Side Panel.
- **Smart Element Handling**: Strips intrusive drop shadows and accurately recolors tables, SVG icons, and text inputs.
- **Google Sheets Support**: Intelligently detects and perfectly darkens Google Sheets cell canvases.

## Installation Guide

1. Download the latest `Eclipse.zip` from [this link](https://github.com/multiverseweb/Eclipse/raw/refs/heads/main/Eclipse.zip).
2. Extract the ZIP file to a safe location on your computer.
3. Open your browser and navigate to `chrome://extensions/`.
4. Enable **Developer mode** at the top right of the page.
5. Click **Load unpacked** and select the extracted `src` folder.
